for the site

The title of the site, about time, is a reference to the 2013 movie of the same name. Since this site is about time. And it's about time that there's a time study site!

I used AngularJS (with UI router) as the backbone of this site. Bootstrap was used as the CSS framework. I drew inspiration from Giga Tamarashvili for the design. The pattern used as a background is from subtlepatterns. There was a ton of stackoverflow use as well.

Note that there are some minor bugs here and there. I didn't have enough time to test it out thoroughly. The design isn't responsive, meaning, I didn't plan it for mobile. That is included in the future plans for this site.
