time study
This is a work measurement technique done by slicing the task into elements and timing each element, establishing ratings, and assigning appropriate allowances to determine time standards. For more info, go here.
elements
In a time study, a task is broken down to elements. For example, in a morning routine, the elements are: arrange bed, eat breakfast, take a shower, and arrange bag.
allowances
This is the extra time allocated for personal interruptions, fatigue, and unavoidable delays.
